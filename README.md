# todo-22-10-2021
A demo todo app for the demonstration of the industrial work

## Setup
**clone**
```js
git clone -b {branch_name} https://github.com/ats1999/todo-22-10-2021.git
```
**go to directory**
```js
cd todo-22-10-2021

// create client
npx create-react-app client

// create server 
mkdir server
cd server

// initialize server
npm init 
```
